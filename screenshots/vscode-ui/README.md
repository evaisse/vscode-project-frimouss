## How These Were Generated

These mockups are automatically generated by the `generate-vscode-ui` script:

```bash
npm run generate-vscode-ui
```

The script:
1. Uses the extension's avatar generation core
2. Creates realistic VSCode UI layouts with HTML/CSS
3. Embeds the generated avatars
4. Applies proper VSCode theme colors
5. Simulates the extension's integration points

## Regenerating Screenshots

To regenerate all VSCode UI mockups:

```bash
npm run compile
npm run generate-vscode-ui
```

This will update all HTML files in this directory with fresh mockups.
